Edwin Aljibe - Personal Profile Website

A simple personal profile website made for CMSC 207 Web Programming and Development course mini project.

---------------------------------------------

A. What This Project Is About

This is a one-page website that introduces me - Edwin Aljibe - as an Architect, Environmental Planner, and cybersecurity-certified professional. 
It shows my credentials, hobbies, and contact information in a clean and organized way.

I built this project to practice:
- Writing proper HTML page structure
- Styling a webpage with CSS
- Adding simple interactivity with JavaScript
- Making the site work on both phones and computers

---------------------------------------------

B. Files Included

- aljibeedwin.github.io/ 
    - index.html (The main webpage)
    - styles.css (All the colors, fonts, and layout styles)
    - script.js (The interactive parts: button, menu, animations)
    - README.md (This file)
    - aljibeedwin.jpg (My profile photo)
    - logos/ (Credential badge images)
        - uap.png (Architect)
        - piep.png (Environmental Planner)
        - leed.png (LEED Green Associate)
        - revit.png (Autodesk Certified Professional)
        - cc.png (Certified in Cybersecurity)

The three main files work together like this: HTML builds the structure, CSS makes it look good, and JavaScript makes it interactive.

---------------------------------------------

C. Technologies Used

- HTML5: for the page structure and content
- CSS3: for styling, layout, and animations
- JavaScript: for interactive features
- Google Fonts: Playfair Display and DM Sans
- GitHub Pages: for hosting the live site

The site was built entirely from scratch without using any external frameworks or libraries.

---------------------------------------------

D. Features

Layout & Structure
- Four sections: Hero (intro), About Me, Hobbies & Interests, and Contact
- Proper HTML tags like header, section, article, and footer
- Alt text on all images for accessibility

Navigation
- Top menu bar with links that jump to each section smoothly
- On mobile, the menu collapses into a hamburger button
- The current section's nav link highlights as you scroll

Hero Section
- Animated intro with my name and a short tagline
- A "Discover a Fun Fact" button that shows a random fact about me each time it's clicked

About Me
- My profile photo with a decorative gold border frame
- A short bio paragraph
- Five credential logo cards (one for each certification) with hover effects

Hobbies & Interests
- Three cards covering Cybersecurity, Design, and Travel
- Each card has a hover animation

Contact
- Links to my Email, GitHub, and Credly profile
- All links open in a new tab

Responsive Design
- The layout adjusts for tablets and mobile phones 
- Elements fade in as you scroll down the page

F. How to Customize It

What you want to change-----(Where to edit it)
- Name, bio, section text   (index.html)
- Colors and fonts		    (root variables at the top of styles.css)
- Fun fact button text		(funFacts array in script.js)
- Profile photo			    (Replace aljibeedwin.jpg with your desired image)
- Credential logos		    (Swap PNG files inside the logos/ folder)
